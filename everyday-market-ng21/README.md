# Everyday Market App - Angular 21

**Student:** Olabisi Afolabi  
**Framework:** Angular 21  
**Date:** May 28, 2026  

## Live Demo Features

- Registration form with validation (name, email, phone, address, province, country, terms)
- Custom validator: Canada only (selecting United States shows error)
- Products page with 5 categories and prices
- Routing between `/register` and `/products`

## How to Run

```bash
npm install
ng serve
Open http://localhost:4200
